Arrays

Arrays are just a fancy term for lists!

Core Structure:

An array is a series of elements which are of the same type. Each element inside the array can be referenced using its unique identifier.
For example, 5 values of type int can be declared as an array without having to declare all 5 seperately.
You can access the individual elements by their index in the array. Note: Arrays are 0 indexed, meaning they start counting at zero!

Initializing Arrays

If you don't initialize an array with pre-defined values, it'll be created with un-initialized elements. This means that none of the elements are set to a specific value.
Elements can be provided when creating the array in order to initialize the array with the predefined values.

int bar[5]; // Five random floating values
int foo[5] = {10, 20, 30, 40, 50}; // Initializes with 5 values

Accessing the elements in an array

The values of any of the elements in an array can be accessed just like any other variable of the same type. You reference the element using the following syntax:

array_name[position_in_array];
Remember, arrays are zero indexed!
So for example, in the above example with 5 elements in an array, you would access the 3rd element using
foo[2]; // 0, 1, 2

Multidimensional Arrays

A Multidimensional array can be simply described as an 'array of arrays'. There is no different syntax for accessing the element in the array, you just provide a second index!

int foo[5][10];
